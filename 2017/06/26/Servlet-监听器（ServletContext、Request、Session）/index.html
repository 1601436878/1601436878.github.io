  • ServletContext — ServletContextAttributeListener

  • attributeAdded () 监听属性的增加

钝化和活化

  • 钝化 将session序列化到磁盘
  • 活化 将序列化的对象反序列化到内存中

什么时候钝化

  • 当tomcat服务器关闭或重启时，会将当前的session对象钝化
  • 当web应用被重新加载时（reload）钝化

当tomcat服务器重启开启时，重新加载文件道内存并删除本地文件
