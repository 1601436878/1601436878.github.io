简介

sudo命令允许系统管理员让普通用户执行一些或者全部的root命令。不需要暴露root密码，就可以让普通用户执行管理员的命令

  • visudo

安全性

如果用sudo给普通用户赋予passwd命令的权限，那么普通用户就可以用sudo命令来修改root密码，这是十分不安全的。不过，在赋予权限时可以加上一些限制，将普通用户可用的命令限制在合理范围内。

  • 可以使用正则表达式，不需要加引号
  • 使用 ! 取反，表示不允许执行这条命令
